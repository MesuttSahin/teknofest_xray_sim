#  Jupyter Notebooks (Deneme Tahtası)

Burası kodları deneme, veriyi analiz etme (EDA) ve prototip oluşturma alanıdır.

##  Kurallar
1. **İsimlendirme Standardı:** Dosyalar `Numara_KısaAcıklama_Yazar.ipynb` şeklinde olmalıdır.
   * ✅ `01_veri_analizi_mert.ipynb`
   * ❌ `deneme.ipynb`, `test.ipynb` (YASAK!)
2. **Commit Etmeden Önce:** Notebook çıktılarını (Output cells) temizleyerek GitHub'a atın. (Dosya boyutu şişmesin).
3. **Burada Kalıcı Kod Yok:** Eğer yazdığınız kod işe yarıyorsa, onu temizleyip `src/` klasörüne taşıyın.
