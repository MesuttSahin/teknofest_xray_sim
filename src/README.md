# Kaynak Kodlar (Source Code)

Projenin **kalbi** burasıdır. Ürünleşecek, temiz ve modüler Python kodları burada tutulur.

##  Modüller
* **`data/`**: Veri yükleme (`dataset.py`) ve işleme (`transforms.py`) kodları.
* **`models/`**: Yapay zeka mimarileri (`model.py`) ve eğitim döngüleri (`train.py`).

##  Kural
Buradaki kodlar `.ipynb` (Notebook) formatında OLAMAZ. Sadece `.py` dosyaları bulunur.
