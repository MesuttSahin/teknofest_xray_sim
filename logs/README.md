#  Loglar (Logs)

Model eğitimi sırasında makine tarafından üretilen kayıtlar buradadır.

##  İçerik
* **`metrics.csv`**: Epoch epoch Loss ve Accuracy değerlerini tutan dosya.
* **`training.log`**: Hata mesajları veya sistem çıktıları.
* **Tensorboard Dosyaları:** (Eğer kullanırsak) `events.out.tfevents...` dosyaları.

**Not:** Bu klasördeki dosyalar genelde `.gitignore` ile yoksayılır ancak final sonuçları (`metrics.csv`) manuel olarak eklenebilir.
